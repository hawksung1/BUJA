# BUJA 프로젝트 문서

이 디렉토리에는 BUJA 프로젝트의 설계 및 계획 문서가 포함되어 있습니다.

## 문서 목록

### 1. REQUIREMENTS.md
**요구사항 명세서**

프로젝트의 기능 요구사항, 비기능 요구사항, 데이터 요구사항 등을 상세히 정의한 문서입니다.

- 기능 요구사항 (LLM Agent, 사용자 관리, 투자 성향, 자산 배분 등)
- 비기능 요구사항 (성능, 보안, 사용성 등)
- 데이터 요구사항
- 통합 요구사항
- 우선순위별 분류

### 2. DESIGN.md
**프로그램 설계서**

시스템 아키텍처, 데이터베이스 설계, 모듈 구조, API 설계 등을 포함한 기술 설계 문서입니다.

- 시스템 아키텍처
- 기술 스택
- 데이터베이스 설계 (ERD, 테이블 스키마)
- 모듈 구조 및 클래스 설계
- API 설계
- 보안 설계
- Python 전문가 검토 의견

### 3. WBS.md
**작업 분해 구조 (Work Breakdown Structure)**

프로젝트를 단계별로 분해한 작업 계획서입니다.

- Phase별 작업 분해
- 작업 ID 및 의존성 관계
- 예상 기간 및 리소스 할당
- 마일스톤
- 리스크 관리

## 문서 읽기 순서

1. **REQUIREMENTS.md** - 프로젝트 요구사항 파악
2. **DESIGN.md** - 시스템 설계 이해
3. **WBS.md** - 개발 계획 및 일정 확인

## 문서 업데이트

각 문서는 프로젝트 진행에 따라 지속적으로 업데이트됩니다. 주요 변경사항은 각 문서의 버전 정보를 확인하세요.
