# BUJA 프로젝트 WBS (Work Breakdown Structure)

## 프로젝트 개요

- **프로젝트명**: BUJA - LLM 기반 자산 관리 및 투자 제안 시스템
- **총 기간**: 약 6개월 (24주)
- **개발 방법론**: 애자일 (2주 스프린트)

---

## WBS 구조

```
BUJA 프로젝트
├── 1. 프로젝트 설정 및 인프라 구축
├── 2. Phase 1: 기본 기능 개발
├── 3. Phase 2: 핵심 기능 개발
├── 4. Phase 3: 고급 기능 개발
└── 5. 배포 및 운영 준비
```

## 7. 리스크 관리

### 7.1 기술적 리스크

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
| LLM API 비용 초과 | 높음 | Rate Limiting, 캐싱 강화, 사용량 모니터링 |

| 데이터베이스 성능 이슈 | 중간 | 조기 성능 테스트, 인덱스 최적화 |

| 리스크 | 영향도 | 대응 방안 |
|--------|--------|----------|
