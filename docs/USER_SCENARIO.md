## 시나리오 요약

### 고객 여정
1. **온보딩**: 기본 정보, 재무 상황, 투자 성향, 목표 설정
2. **초기 상담**: 투자 계획 수립 및 포트폴리오 추천
3. **투자 실행**: 실제 증권사에서 투자 시작
4. **정기 모니터링**: 월별/분기별 성과 확인 및 조정
5. **장기 관리**: 목표 달성까지 지속적인 관리

### LLM의 역할
- **맞춤형 조언**: 사용자 정보 기반 개인화된 투자 전략 제시
- **도구 활용**: 포트폴리오 분석, 리스크 계산, 추천 생성 Tool 사용
- **실용적 정보**: 증권사 추천, 수수료 정보, 실행 방법 안내
- **지속적 모니터링**: 정기적인 성과 분석 및 전략 조정 제안

### 고객 가치
- **명확한 목표 설정**: 5년 후 주택 구매 목표 구체화
- **체계적 투자 계획**: 단계별 투자 전략 수립
- **지속적 관리**: 정기적인 상담을 통한 포트폴리오 최적화
- **정보 기반 의사결정**: 데이터 기반 투자 조언 제공
