# 위대한 투자자들의 철학 — BUJA 지식 베이스

이 문서는 BUJA 자산관리 에이전트의 조언 근거가 되는 투자 철학 모음입니다.

---

## 1. 워런 버핏 (Warren Buffett)

### 핵심 철학
- **경제적 해자(Economic Moat)**: 경쟁자가 넘어오기 어려운 독점적 우위를 가진 기업만 투자
- **안전마진(Margin of Safety)**: 내재가치보다 충분히 낮은 가격에 매수
- **장기 보유**: "좋은 주식을 사고 아무것도 하지 마라" — 복리의 마법
- **사업 이해**: 10년 후에도 이 사업이 살아있을지 이해할 수 없으면 투자하지 마라

### 핵심 지표
- ROE(자기자본이익률) 15% 이상 지속
- 부채비율 낮음
- 영업이익률 안정적
- 자유현금흐름(FCF) 창출 능력

### 명언
> "남들이 탐욕스러울 때 두려워하고, 남들이 두려워할 때 탐욕스러워라."
> "10년 보유할 자신이 없으면 10분도 보유하지 마라."
> "시간은 훌륭한 사업의 친구이고, 나쁜 사업의 적이다."

### 투자 체크리스트
- [ ] 이 사업을 충분히 이해하는가?
- [ ] 경쟁사가 쉽게 따라올 수 없는 해자가 있는가?
- [ ] 경영진이 정직하고 유능한가?
- [ ] 주가가 내재가치보다 낮은가?
- [ ] 10년 후 더 강해질 사업인가?

## 2. 벤저민 그레이엄 (Benjamin Graham)

### 핵심 철학
- **내재가치(Intrinsic Value)**: 기업의 자산, 수익력, 배당 등을 기반으로 계산한 진정한 가치
- **Mr. Market**: 시장은 매일 다른 가격을 제시하는 감정적인 파트너 — 그 감정에 휩쓸리지 마라
- **방어적 투자자 vs 공격적 투자자**: 본인의 성향에 맞는 전략 선택
- **분산투자**: 최소 10~30개 종목 분산

### 핵심 지표
- P/E 비율: 15배 이하
- P/B 비율: 1.5배 이하
- 유동비율(Current Ratio): 2.0 이상
- 장기 부채: 순유동자산 이하
- EPS 성장: 최근 10년간 33% 이상 성장

### 명언
> "투자는 분석을 통해 원금의 안전과 적절한 수익을 약속하는 것이다."
> "시장은 단기적으로 투표 기계이지만, 장기적으로는 저울이다."

## 3. 피터 린치 (Peter Lynch)

### 핵심 철학
- **10-bagger**: 10배 오를 종목을 일상에서 발견하라
- **자신이 아는 것에 투자**: 소비자로서 먼저 경험한 좋은 제품/서비스의 회사
- **기업 분류**: 저성장주, 대형우량주, 고성장주, 경기순환주, 회생주, 자산주
- **성장과 가격의 균형**: PEG 비율 = P/E ÷ 성장률, 1.0 이하가 매력적

### PEG 비율 해석
| PEG | 해석 |
|-----|------|
| < 0.5 | 극도의 저평가 — 강력 매수 신호 |
| 0.5 ~ 1.0 | 저평가 — 매수 고려 |
| 1.0 ~ 2.0 | 적정 가치 |
| > 2.0 | 고평가 — 주의 필요 |

### 명언
> "자신이 모르는 것에 투자하면 도박이다."
> "주식 시장의 하락은 재난이 아니라 할인 행사다."
> "당신이 가장 잘 아는 산업에서 10-bagger를 찾아라."

### 투자 체크리스트
- [ ] 이 기업의 제품/서비스를 직접 사용해봤는가?
- [ ] PEG 비율이 1.0 이하인가?
- [ ] 기관투자자 비중이 낮아 아직 주목받지 못했는가?
- [ ] 회사 이름이 지루하고 사업이 단순한가? (좋은 신호)
- [ ] 내부자(임원)들이 자사 주식을 매수하고 있는가?

## 4. 레이 달리오 (Ray Dalio)

### 핵심 철학
- **올웨더 포트폴리오(All Weather Portfolio)**: 경제 사이클의 4계절 모두에서 살아남는 자산 배분
- **리스크 패리티(Risk Parity)**: 금액이 아닌 리스크를 균등하게 배분
- **부채 사이클**: 단기(5~8년)와 장기(50~75년) 부채 사이클 이해

### 올웨더 포트폴리오 기본 배분
| 자산 | 비중 |
|------|------|
| 미국 주식 | 30% |
| 장기 국채(20~25년) | 40% |
| 중기 국채(7~10년) | 15% |
| 금 | 7.5% |
| 원자재 | 7.5% |

### 경제 사이클 4계절
| 계절 | 경제 상황 | 유리한 자산 |
|------|-----------|-------------|
| 봄 | 성장↑ 인플레↑ | 주식, 원자재, 금 |
| 여름 | 성장↓ 인플레↑ | 인플레 연동채권, 원자재, 금 |
| 가을 | 성장↑ 인플레↓ | 주식, 채권 |
| 겨울 | 성장↓ 인플레↓ | 채권, 현금 |

### 명언
> "다각화는 투자의 성배다."
> "고통 + 반성 = 성장"
> "원칙에 따라 행동하라, 감정에 따라 행동하지 마라."

## 5. 필립 피셔 (Philip Fisher)

### 핵심 철학
- **성장주 장기 투자**: 뛰어난 경영진과 강력한 성장 잠재력을 가진 기업에 장기 투자
- **스커틀버트(Scuttlebutt)**: 공급업체, 고객, 경쟁사, 전직 직원과의 대화로 기업 실체 파악
- **소수 집중 투자**: 10~12개 이상 분산하지 말 것 — 가장 좋은 것에 집중

### 15가지 투자 기준 (요약)
1. 향후 수년간 매출 성장 가능성
2. 경영진의 신제품/서비스 개발 의지
3. R&D 효율성
4. 평균 이상의 영업조직
5. 충분한 이익률
6. 이익률 개선 노력
7. 우수한 노사관계
8. 탁월한 임원 관계
9. 경영진 깊이(depth of management)
10. 원가/재무 분석 능력
11. 경쟁사 대비 차별화 요소
12. 단기 이익보다 장기 성장 우선 여부
13. 성장 자금 조달 시 주주 희석 최소화
14. 어려울 때 숨기지 않는 경영진 투명성
15. 경영진의 도덕성

### 명언
> "위대한 기업의 주식은 절대 팔지 마라."
> "주식시장은 조급한 사람의 돈을 인내심 있는 사람에게 전달하는 장치다."

## 분석 프레임워크: BUJA 어드바이저 의사결정 트리

```
종목 분석 요청
│
├─ 1단계: 사업 이해 (버핏 기준)
│   ├─ 이해 가능? → 계속
│   └─ 이해 불가? → "이 종목은 분석 범위를 벗어납니다"
│
├─ 2단계: 밸류에이션 (그레이엄 기준)
│   ├─ P/E < 15, P/B < 1.5? → 저평가 신호
│   └─ 고평가? → 성장률로 정당화 가능한지 확인
│
├─ 3단계: 성장성 (린치/피셔 기준)
│   ├─ PEG < 1.0? → 성장 대비 저평가
│   └─ 10-bagger 가능성? → 시장 규모와 성장 동력 확인
│
├─ 4단계: 리스크 (달리오 기준)
│   ├─ 매크로 환경은?
│   └─ 포트폴리오 내 비중은 적절한가?
│
└─ 5단계: 최종 판단
    ├─ 매수 / 보유 / 매도
    └─ 적정 비중 제안
```
