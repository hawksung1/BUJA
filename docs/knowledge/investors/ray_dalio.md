# 레이 달리오 — 올웨더·리스크패리티·부채사이클·매크로

## 핵심 철학

> "가장 중요한 것은 잘못될 수 있다는 것을 인정하는 것이다."
> "다각화는 투자의 성배(Holy Grail)다. 서로 상관관계가 낮은 15~20개 자산을 보유하면 리스크를 80% 줄이면서 수익은 그대로 유지할 수 있다."
> "경제는 기계처럼 작동한다. 패턴은 반복된다."

달리오는 브리지워터를 세계 최대 헤지펀드로 키운 매크로 투자자이자 시스템 사상가다. 그의 접근은 철저히 **원칙 기반, 데이터 기반, 시나리오 기반**이다.

---

## 1. 경제 기계 — 4계절 프레임

달리오는 경제를 두 가지 축으로 분류한다:

```
              성장 UP
                 │
     [봄]        │      [여름]
  성장 ↑ 인플레↓  │  성장 ↑ 인플레↑
                 │
인플레 DOWN ─────┼───── 인플레 UP
                 │
     [겨울]       │      [가을]
  성장 ↓ 인플레↓  │  성장 ↓ 인플레↑
                 │
              성장 DOWN
```

### 계절별 최적 자산군

| 계절 | 환경 | 유리한 자산 |
|------|------|-------------|
| **봄** | 성장 ↑, 인플레 ↓ | 주식, 회사채 |
| **여름** | 성장 ↑, 인플레 ↑ | 원자재, 금, 물가연동채(TIPS) |
| **가을** | 성장 ↓, 인플레 ↑ | 금, 원자재, 단기채 (스태그플레이션) |
| **겨울** | 성장 ↓, 인플레 ↓ | 장기채, 현금, 금 |

## 2. 올웨더 포트폴리오

어떤 경제 환경에서도 안정적인 수익을 내도록 설계된 자산 배분:

```
올웨더 포트폴리오 (달리오 공개 버전)
├── 장기 미국 국채 (20년+)    40%
├── 주식 (S&P500 등)          30%
├── 중기 미국 국채 (7~10년)   15%
├── 금                         7.5%
└── 원자재 (DJP 등)            7.5%
```

### 핵심 원리: 리스크 패리티
- 전통 60/40 포트폴리오는 주식 리스크가 **90%** 차지
- 올웨더는 각 자산군의 **리스크 기여도**를 균등하게 배분
- 명목 비중이 아닌 **변동성 조정 비중**으로 구성

### 한국 투자자를 위한 적용
```
올웨더 한국 버전 (참고)
├── 국내외 주식 (KODEX200, S&P500 ETF)   30%
├── 장기 국채 (30년 국고채, TLT)          30%
├── 중기 국채 (10년 국고채, IEF)          15%
├── 금 (KODEX 골드선물, GLD)              15%
└── 원자재 (KODEX 에너지화학, DBC)        10%
```

## 3. 부채 사이클 이론

달리오의 핵심 매크로 프레임워크.

### 단기 부채 사이클 (5~8년)
```
신용 팽창 → 경기 호황 → 인플레이션 →
중앙은행 금리 인상 → 신용 수축 → 경기 침체 →
금리 인하 → 다시 신용 팽창 (반복)
```

### 장기 부채 사이클 (75~100년)
```
초기: 생산성 기반 성장 (건전)
중기: 부채 기반 성장 (취약)
말기: 부채 상환 불능 → 디레버리징

디레버리징 4가지 방법:
1. 긴축 (지출 삭감) — 디플레이션 압력
2. 채무 불이행/구조조정 — 디플레이션 압력
3. 부의 재분배 (세금) — 사회 갈등
4. 화폐 발행 (양적완화) — 인플레이션 압력
```

### "Beautiful Deleveraging" (아름다운 디레버리징)
- 4가지 방법을 균형 있게 혼합할 때 발생
- 명목 성장률 > 명목 금리 유지 필요
- 2008~2015년 미국이 이를 달성 (달리오 평가)

## 4. 달리오가 보는 주요 매크로 지표

| 지표 | 관찰 포인트 |
|------|------------|
| **실질 금리** | 마이너스면 금·원자재 유리, 플러스면 달러·채권 유리 |
| **수익률 곡선** | 역전(장단기 금리차 마이너스) = 경기 침체 선행 신호 |
| **신용 스프레드** | 확대 = 리스크 오프, 축소 = 리스크 온 |
| **통화 공급(M2)** | 급증 = 인플레이션 선행, 급감 = 디플레이션 선행 |
| **부채/GDP** | 장기 사이클 위치 판단 |
| **경상수지** | 기축통화국 지위 유지 여부 |

## 5. 리스크 관리 원칙

### 상관관계 분산
```
포트폴리오 리스크 감소 = 자산 추가 × (1 - 상관관계)

예시:
- 주식 + 주식: 상관관계 0.8 → 리스크 거의 안 줄어듦
- 주식 + 장기채: 상관관계 -0.3 → 리스크 크게 감소
- 주식 + 금: 상관관계 0.0~0.2 → 리스크 감소
```

### 달리오의 5% 규칙
- 단일 자산에 **절대 5% 이상 집중** 금지 (리스크 기준)
- "당신이 얼마나 확신하든 틀릴 수 있다"

## 6. 현재 경제 환경 진단 프레임

분석 요청 시 아래 순서로 진단:

```
1. 현재 경제 계절은? (성장률 방향 + 인플레이션 방향)
2. 단기 부채 사이클 어디에 위치? (팽창/수축)
3. 장기 부채 사이클 어디에 위치? (초기/중기/말기)
4. 중앙은행 정책 방향? (긴축/완화)
5. 달러 사이클? (강달러/약달러)
6. 각 자산군 유불리 판단
7. 포트폴리오 조정 방향
```

## 7. 달리오 언어로 분석할 때 사용할 표현

- "현재 경제 사이클은 달리오 4계절 기준 [계절]에 위치합니다. 근거: [성장률 지표], [인플레이션 지표]"
- "장기 부채 사이클상 [초기/중기/말기] 단계로, [리스크 내용]에 주의가 필요합니다."
- "올웨더 기준으로 현재 포트폴리오는 [자산군]에 과중되어 있습니다."
- "수익률 곡선이 [정상/역전] 상태로, 향후 [X]개월 내 경기 침체 가능성을 시사합니다."
- "실질 금리가 [X]%로 [플러스/마이너스]이므로 금·원자재는 [유리/불리]합니다."
