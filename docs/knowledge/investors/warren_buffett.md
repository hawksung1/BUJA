# 워런 버핏 — 가치투자·경제적 해자·장기보유

## 핵심 철학

> "훌륭한 회사를 적정 가격에 사는 것이, 적정 회사를 훌륭한 가격에 사는 것보다 훨씬 낫다."
> "우리가 선호하는 보유 기간은 영원이다."
> "10년을 보유할 자신이 없다면, 10분도 보유하지 마라."

버핏의 투자는 **비즈니스 오너십** 개념에서 출발한다. 주식을 사는 것은 회사의 일부를 사는 것이며, 그 회사가 앞으로 수십 년간 얼마나 많은 현금을 벌어들일 수 있느냐가 핵심이다.

---

## 1. 경제적 해자 (Economic Moat)

버핏이 가장 중시하는 개념. 경쟁자가 쉽게 침범할 수 없는 구조적 우위.

### 해자의 4가지 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **브랜드 파워** | 소비자가 프리미엄을 기꺼이 지불하는 브랜드 | 코카콜라, 애플 |
| **전환 비용** | 다른 제품으로 바꾸기 어려운 구조 | Oracle DB, Bloomberg 터미널 |
| **네트워크 효과** | 사용자가 늘수록 가치가 높아지는 플랫폼 | Visa, 카카오 |
| **원가 우위** | 구조적으로 낮은 비용 구조 | 코스트코, 아마존 물류 |

### 해자 판별 질문
- 이 회사가 가격을 10% 올려도 고객이 떠나지 않는가?
- 10년 후에도 이 사업 모델이 유효한가?
- 경쟁사가 10억 달러를 투자해도 이 회사를 따라잡기 어려운가?

## 2. 내재가치와 안전마진

### 내재가치 계산 (DCF 기반)
버핏은 공식을 공개하지 않지만, 기본 원리는 **향후 기업이 생산할 모든 현금흐름의 현재가치**다.

```
내재가치 ≈ FCF × (1 - g/r) / (r - g)

FCF: 잉여현금흐름 (영업현금흐름 - 설비투자)
g: 장기 성장률 (보수적으로 3~7%)
r: 할인율 (10년물 국채 + 리스크 프리미엄, 보통 8~12%)
```

### 안전마진 (Margin of Safety)
- **최소 안전마진**: 내재가치 대비 **25~30% 이상** 할인된 가격
- 불확실성이 클수록 더 큰 안전마진 요구
- "맞는 것보다 안전한 것이 더 중요하다"

## 3. 핵심 재무 지표

| 지표 | 버핏 기준 | 의미 |
|------|-----------|------|
| **ROE** | 15% 이상 (10년 평균) | 자기자본 효율성 |
| **ROA** | 10% 이상 | 자산 효율성 |
| **영업이익률** | 20% 이상 (경쟁사 대비 우위) | 가격 결정력 |
| **부채비율** | 0.5 이하 또는 낮은 추세 | 재무 건전성 |
| **FCF Yield** | 4% 이상 | 현금 창출력 vs 시가총액 |
| **PER** | 업종 평균 이하, 단 성장성 고려 | 수익 대비 가격 |
| **자사주 매입** | 내재가치 이하일 때만 | 주주환원 건전성 |

## 4. 경영진 평가

버핏은 재무제표만큼 경영진을 중시한다.

### 좋은 경영진의 특징
- **자본 배분 능력**: ROE가 높을 때 재투자하고, 낮을 때 배당·자사주 매입
- **주주 친화적 커뮤니케이션**: 실수를 솔직하게 인정하는 주주서한
- **사업 집중**: 이해하지 못하는 분야로 문어발 확장 안 함
- **적은 스톡옵션**: 경영진 보상이 주주 이익과 정렬

### 나쁜 경영진 신호
- IR자료에 EBITDA만 강조 (감가상각 숨기기)
- 잦은 M&A (자본 배분 실패 징후)
- 경쟁사 탓, 거시경제 탓만 하는 커뮤니케이션

## 5. 투자 결정 체크리스트

```
✅ 이 사업을 나는 완전히 이해하는가?
✅ 10년 후에도 이 사업이 존재할 것인가?
✅ 지속 가능한 경쟁 우위(해자)가 있는가?
✅ 경영진을 신뢰할 수 있는가?
✅ 현재 가격이 내재가치 대비 25% 이상 싼가?
✅ 재무 상태가 건전한가? (부채가 적정한가?)
✅ ROE가 지속적으로 15% 이상인가?
✅ FCF를 꾸준히 창출하는가?
```

## 6. 버핏이 피하는 것

- **예측 불가능한 사업**: 반도체 사이클, 원자재 가격 의존 기업
- **과도한 부채**: 경기 침체 시 생존 불확실
- **이해 못하는 기술**: 초기 인터넷·바이오 기업 기피 (단, 애플은 소비자 브랜드로 이해)
- **적자 성장 기업**: 언젠가 수익이 난다는 기대만으로 투자 금지
- **유행 따라가기**: 닷컴 버블, 암호화폐 등

- "ROE가 [X]%로 버핏 기준 15%를 [상회/하회]합니다."
- "경영진의 자본 배분을 보면 [관찰 내용]이 [긍정/우려] 신호입니다."
