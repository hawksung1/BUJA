# 벤저민 그레이엄 — 내재가치·안전마진·Mr. Market

## 핵심 철학

> "단기적으로 시장은 투표 기계이지만, 장기적으로는 저울이다."
> "투자란 철저한 분석을 통해 원금의 안전과 적절한 수익을 보장받는 행위다. 이 조건을 충족하지 못하면 투기다."
> "Mr. Market은 당신의 주인이 아니라 당신의 하인이다."

그레이엄은 감정을 배제한 **정량 분석**으로 저평가 주식을 발굴하는 방법론을 체계화했다. 버핏의 스승이자 가치투자의 창시자.

---

## 1. Mr. Market 개념

시장을 **조울증 환자 파트너**로 의인화한 비유.

- 매일 아침 당신의 지분을 사겠다/팔겠다고 가격을 제시함
- 기분 좋은 날: 지나치게 높은 가격 제시 → **매도 기회**
- 기분 나쁜 날: 터무니없이 낮은 가격 제시 → **매수 기회**
- 핵심: Mr. Market의 가격에 감정적으로 반응하면 안 됨. 내재가치를 알아야 Mr. Market을 이용할 수 있음

### 실전 적용
```
시장이 공포에 빠졌을 때 → 그레이엄은 매수 목록 점검
시장이 탐욕에 빠졌을 때 → 그레이엄은 매도·현금 비중 확대
```

## 2. 내재가치 계산법

### 그레이엄 공식 (1962년 버전)
```
내재가치 = EPS × (8.5 + 2g)

EPS: 주당순이익
8.5: 무성장 기업의 적정 P/E
g: 향후 7~10년 예상 연간 성장률(%)
```

### 그레이엄 수정 공식 (금리 조정)
```
내재가치 = EPS × (8.5 + 2g) × (4.4 / Y)

Y: 현재 AAA 등급 회사채 금리(%)
4.4: 그레이엄 시대 기준 금리
```

### NCAV (순유동자산가치) — 극단적 저평가 기준
```
NCAV = 유동자산 - 총부채
NCAV per share = NCAV / 발행주식수

매수 조건: 주가 < NCAV per share × 2/3
```
이 기준을 통과한 주식을 "그레이엄 넷-넷"이라 부름. 매우 보수적인 청산가치 기반 분석.

## 3. 7가지 안전 기준 (방어적 투자자)

그레이엄이 《현명한 투자자》에서 제시한 체크리스트:

| # | 기준 | 세부 조건 |
|---|------|-----------|
| 1 | **기업 규모** | 매출 1억 달러 이상 (소형주 리스크 회피) |
| 2 | **재무 건전성** | 유동비율 2 이상, 장기부채 < 순운전자본 |
| 3 | **배당 지속성** | 최소 20년 연속 배당 지급 |
| 4 | **적자 없음** | 최근 10년간 연속 적자 없음 |
| 5 | **이익 성장** | 10년간 EPS 최소 1/3 이상 증가 |
| 6 | **P/E 기준** | P/E ≤ 15 (최근 3년 평균 EPS 기준) |
| 7 | **P/B 기준** | P/B ≤ 1.5, 단 P/E × P/B ≤ 22.5 |

## 4. 핵심 밸류에이션 지표

| 지표 | 그레이엄 기준 | 판단 |
|------|--------------|------|
| **P/E** | 15 이하 | 시장 평균 P/E와 비교 |
| **P/B** | 1.5 이하 | 자산 대비 가격 |
| **P/E × P/B** | 22.5 이하 | 복합 저평가 기준 |
| **유동비율** | 2.0 이상 | 단기 지급능력 |
| **부채비율** | 1.0 이하 | 장기 재무 안정성 |
| **배당수익률** | 시장평균 이상 | 수익 실현 안전망 |

---

## 5. 안전마진 (Margin of Safety)

그레이엄의 핵심 원칙. 내재가치보다 **충분히 싸게** 사야 한다.

```
안전마진 = (내재가치 - 시장가격) / 내재가치 × 100%

그레이엄 최소 기준: 33% 이상
(예: 내재가치 100원 → 67원 이하에서만 매수)
```

왜 안전마진이 필요한가?
- 내재가치 계산 자체가 틀릴 수 있음
- 미래는 예측 불가능
- 안전마진은 실수를 용납하는 버퍼

## 6. 투기 vs 투자 구분

그레이엄이 강조한 핵심 구분:

| 투자 | 투기 |
|------|------|
| 철저한 분석 기반 | 시세 차익 기대 |
| 원금 안전 확보 | 원금 손실 감수 |
| 적절한 수익 추구 | 과도한 수익 추구 |
| 내재가치 근거 매수 | 가격 상승 기대 매수 |
