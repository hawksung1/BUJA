# 5대 투자자 철학 — 분석 기준

## 1. 워런 버핏 — 가치투자·경제적 해자

**핵심 원칙**
- 경쟁자가 침범할 수 없는 경제적 해자(브랜드·전환비용·네트워크효과·원가우위) 보유 기업
- "훌륭한 회사를 적정 가격에" — 내재가치 대비 25% 이상 할인 시 매수
- 보유 기간은 영구. ROE 15% 이상, FCF 꾸준히 창출

**핵심 판정 기준**
| 지표 | 기준 |
|------|------|
| ROE | 15% 이상 (10년 평균) |
| 영업이익률 | 20% 이상 |
| 부채비율 | 0.5 이하 |
| FCF Yield | 4% 이상 |
| 안전마진 | 내재가치 대비 25% 이상 할인 |

**피하는 것**: 예측 불가 사업, 과도한 부채, 이해 못하는 기술, 적자 성장주

---

## 2. 벤저민 그레이엄 — 내재가치·안전마진·Mr. Market

**핵심 원칙**
- Mr. Market(조울증 시장)을 이용하라 — 공포 시 매수, 탐욕 시 매도
- 안전마진 33% 이상: 내재가치의 2/3 이하 가격에서만 매수
- 투자 = 철저한 분석 + 원금 안전 확보. 나머지는 투기

**핵심 판정 기준**
| 지표 | 기준 |
|------|------|
| P/E | 15 이하 |
| P/B | 1.5 이하 |
| P/E × P/B | 22.5 이하 |
| 유동비율 | 2.0 이상 |
| 그레이엄 공식 내재가치 | EPS × (8.5 + 2g) × (4.4/금리) |
| 안전마진 | 33% 이상 |

**7가지 안전 기준**: 기업규모·재무건전성·20년 배당·적자없음·이익성장·P/E≤15·P/B≤1.5

## 3. 피터 린치 — 성장주·PEG·10-bagger·현장조사

**핵심 원칙**
- "자신이 아는 것에 투자하라" — 일상에서 성장 기업을 먼저 발견
- PEG(P/E ÷ 성장률) 1.0 이하가 매수 기준
- 6분류 체계: 대형저성장·우량주·고성장·경기민감·자산주·회생주

**핵심 판정 기준**
| 지표 | 기준 |
|------|------|
| PEG | 1.0 이하 (0.5 이하면 강력 매수) |
| 부채비율 | 25% 이하 |
| 매출 성장률 | 20% 이상 (고성장주 기준) |
| 재고 증가율 | 매출 증가율 이하 (초과 시 경고) |
| 기관 보유 | 5% 미만이면 아마추어 어드밴티지 |

**10-bagger 후보 특징**: 소형주, 지루한 이름, 틈새 독점, 월가 무관심, 내부자 매수

## 4. 레이 달리오 — 올웨더·리스크패리티·부채사이클·매크로

**핵심 원칙**
- 경제를 4계절로 분류: 성장↑인플레↓(봄)·성장↑인플레↑(여름)·성장↓인플레↑(가을)·성장↓인플레↓(겨울)
- 올웨더: 주식30%·장기채40%·중기채15%·금7.5%·원자재7.5%
- 15~20개 비상관 자산으로 리스크 80% 감소 가능

**경제 계절별 유리한 자산**
| 계절 | 환경 | 유리한 자산 |
|------|------|-------------|
| 봄 | 성장↑ 인플레↓ | 주식, 회사채 |
| 여름 | 성장↑ 인플레↑ | 원자재, 금, TIPS |
| 가을 | 성장↓ 인플레↑ | 금, 원자재 (스태그플레이션) |
| 겨울 | 성장↓ 인플레↓ | 장기채, 현금, 금 |

**핵심 매크로 지표**: 실질금리·수익률곡선·신용스프레드·M2·부채/GDP

## 5. 필립 피셔 — 성장기업·15포인트·Scuttlebutt·장기보유

**핵심 원칙**
- "위대한 기업은 절대 팔지 마라" — 성장 스토리가 훼손되지 않는 한 영구 보유
- 15포인트 체크리스트: 성장성·R&D 효율·영업 조직·이익률·노사관계·경영 깊이·장기 전망·경영진 투명성 등
- Scuttlebutt: 경쟁사·고객·납품업체·전직 직원에게 직접 조사

**15포인트 핵심 항목 (판정 기준)**
| 항목 | 확인 질문 |
|------|-----------|
| 판매 성장 | 향후 몇 년간 매출 크게 늘릴 제품·서비스가 있는가? |
| R&D 효율 | R&D 투자 대비 실질 상업 성과가 나오는가? |
| 이익률 수준 | 업종 내 최상위권, 개선 추세인가? |
| 경영 깊이 | CEO 1인 의존이 아닌 두터운 관리층이 있는가? |
| 경영진 투명성 | 나쁜 소식도 솔직하게 공유하는가? |
| 장기 전망 | 단기 이익보다 장기 성장에 투자하는가? |

**매도 3원칙**: ①투자 실수 판명 ②기업 본질 변질 ③더 좋은 기회 발견
