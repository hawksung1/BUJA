# /analyze — 종목 심층 분석

주어진 종목($ARGUMENTS)에 대해 BUJA 어드바이저로서 심층 분석을 수행합니다.

## 분석 프로세스

1. **실시간 데이터 수집**
   - `src/data/stock_fetcher.py`를 사용해 현재 주가, 거래량, 52주 고/저가 조회
   - yfinance로 재무제표(P/E, P/B, ROE, 부채비율, FCF) 수집

2. **뉴스 수집**
   - 최근 7일 관련 뉴스 Top 5 수집
   - 긍정/부정 센티먼트 분류

3. **투자 철학 적용**
   - `docs/knowledge/investment_philosophy.md` 참조
   - 버핏: 해자, 경영진, 이해 가능성
   - 그레이엄: P/E, P/B, 내재가치
   - 린치: PEG, 성장성, 10-bagger 가능성
   - 달리오: 매크로 환경, 포트폴리오 비중

4. **출력 형식**
```
## [종목명] ([티커]) 분석 리포트
📅 분석일: YYYY-MM-DD

### 현재 상황
- 현재가: XXX원 (전일比 +X.X%)
- 거래량: X만주 (평균比 X%)
- 52주 범위: XXX ~ XXX

### 밸류에이션
| 지표 | 값 | 업종 평균 | 평가 |
|------|-----|-----------|------|
| P/E  | X배 | X배       | 저평가/적정/고평가 |
| P/B  | X배 | X배       | ... |
| ROE  | X%  | X%        | ... |
| PEG  | X.X | -         | ... |

### 투자자별 관점
🔵 버핏 관점: ...
🟡 그레이엄 관점: ...
🟢 린치 관점: ...
⚪ 달리오 관점: ...

### 리스크 요인
- 단기: ...
- 중기: ...
- 장기: ...

### 최신 뉴스
1. [뉴스 제목] — 요약
2. ...

### 어드바이저 결론
**판단**: 매수 / 보유 / 매도 / 관망
**적정 비중**: 포트폴리오의 X%
**목표가**: XXX원 (X% 업사이드)

⚠️ 본 분석은 투자 조언이 아니며 참고용입니다.
```
