# /news — 오늘의 주요 투자 뉴스 요약

실시간 뉴스를 수집하여 투자자 관점에서 요약하고 시장 영향을 분석합니다.

## 프로세스

1. **뉴스 수집**
   - NewsAPI로 오늘의 주요 금융/경제 뉴스 수집
   - 네이버 금융 RSS 피드 파싱
   - 키워드: 주식, 금리, 환율, 유가, 반도체, AI, 배터리, 금융정책

2. **분석 및 분류**
   - 시장 영향도: 상(빨강) / 중(노랑) / 하(초록)
   - 투자 관련성 필터링
   - 긍정/부정 뉴스 구분

3. **출력 형식**
```
## 오늘의 투자 뉴스 브리핑
📅 YYYY-MM-DD | 작성: BUJA 어드바이저

### 🔴 주요 이슈 (시장 영향 높음)
1. **[뉴스 제목]**
   - 요약: ...
   - 투자 영향: ...
   - 관련 종목: [종목명]

### 🟡 주목할 뉴스
2. ...

### 🟢 참고 뉴스
3. ...

### 오늘의 시장 온도
전반적 분위기: 매우 긍정 / 긍정 / 중립 / 부정 / 매우 부정
핵심 키워드: #금리 #반도체 #환율

### 어드바이저 코멘트
오늘 주목해야 할 포인트: ...

⚠️ 뉴스 분석은 투자 참고용이며 투자 결정의 책임은 본인에게 있습니다.
```
