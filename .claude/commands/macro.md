# /macro — 거시경제 현황 브리핑

`market_scout` 에이전트를 호출하여 달리오 부채 사이클 프레임워크로 거시경제를 분석합니다.

## 데이터 소스
- FRED API: 미국 금리·CPI·GDP·실업률
- 한국은행 API: 기준금리·환율·국내 경제지표
- 실시간 환율: USD/KRW·EUR/KRW·JPY/KRW

## 분석 구조
1. 현재 경제 사이클 판단 (봄/여름/가을/겨울)
2. 달리오 단기·장기 부채 사이클 위치 진단
3. 섹터별 투자 환경 평가
4. 포트폴리오 조정 제안

출력 형식은 `market_scout` 에이전트 정의를 따름
