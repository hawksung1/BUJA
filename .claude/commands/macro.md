# /macro — 거시경제 현황 브리핑

레이 달리오의 부채 사이클 프레임워크를 기반으로 현재 거시경제 상황을 분석합니다.

## 데이터 소스
- FRED API: 미국 금리, CPI, GDP, 실업률
- 한국은행 API: 기준금리, 환율, 국내 경제지표
- 실시간 환율: USD/KRW, EUR/KRW, JPY/KRW

## 출력 형식
```
## 거시경제 브리핑
📅 YYYY-MM-DD | BUJA 어드바이저

### 현재 경제 사이클: [봄 / 여름 / 가을 / 겨울]

### 핵심 지표
| 지표 | 현재값 | 전월比 | 방향 |
|------|--------|--------|------|
| 미국 기준금리 | X.X% | - | → |
| 한국 기준금리 | X.X% | - | ↑ |
| 미국 CPI(YoY) | X.X% | +X.X%p | ↑ |
| USD/KRW | X원 | +Xp | ↑ |
| 코스피 | X,XXX | +X.X% | ↑ |
| S&P 500 | X,XXX | +X.X% | ↑ |
| WTI 유가 | $XX | +X% | ↑ |
| 금 | $X,XXX | +X% | ↑ |

### 달리오 부채 사이클 위치
단기 사이클: [확장 / 정점 / 수축 / 바닥]
장기 사이클: [초기 / 중기 / 말기]

### 섹터별 투자 환경
| 섹터 | 현재 환경 | 권장 비중 |
|------|-----------|-----------|
| 주식 | 중립 | 유지 |
| 장기채권 | 긍정 | 확대 |
| 금 | 긍정 | 유지 |
| 원자재 | 부정 | 축소 |

### 투자 전략 제안
현재 사이클에서의 권장 포트폴리오 조정: ...

⚠️ 거시경제 분석은 투자 참고용이며 시장은 예측 불가합니다.
```
